# Concept Book References

Pasta para organizar referencias recebidas durante a producao do concept book.

Use subpastas por categoria quando imagens forem salvas localmente:

```text
01_cenarios/
02_personagens/
03_historia/
04_assets_props/
05_ui_hud/
06_editorial_layout/
```

Referencias externas, como Pinterest, devem ser registradas primeiro em:

```text
docs/03_VisualDevelopment/concept-book-reference-intake.md
```

Nao salvar imagens sem origem ou sem contexto de uso.
