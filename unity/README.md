# Unity

Esta pasta esta reservada para o projeto Unity real.

Quando a producao entrar nessa etapa, criar o projeto em:

`unity/FragmentosDoAmanha/`

Nao adicionar manualmente `Assets/`, `Packages/` ou `ProjectSettings/` falsos antes de criar o projeto pela Unity. A estrutura esperada esta documentada em `docs/08_UnityPipeline/unity-production-pipeline.md`.

Proximo passo quando a Unity estiver instalada:

1. Instalar/confirmar Unity 6 LTS.
2. Instalar Git LFS.
3. Criar o projeto pela Unity Hub em `unity/FragmentosDoAmanha/`.
4. Configurar URP 2D, Input System, Pixel Perfect Camera, Cinemachine, Tilemap e 2D Animation.
5. Fazer o commit `chore: add Unity base project`.

Checklist detalhado: `docs/08_UnityPipeline/unity-base-project-setup.md`.
