# Art

Estrutura para organizar producao visual do projeto.

- `illustration/` - concept art, key art, artbook e estudos detalhados.
- `pixel/` - sprites, tilesets e estudos in-game.
- `animation/` - spritesheets, rigs, poses-chave e testes de movimento.
- `ui/` - HUD, menus, icones e elementos de interface.
- `fx/` - efeitos visuais de gameplay.
- `marketing/` - imagens de apresentacao, press kit e materiais publicos.

Assets gerados por IA devem ser tratados como base de trabalho e passar por validacao manual antes de entrar no jogo.
