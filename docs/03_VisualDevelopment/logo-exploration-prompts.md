# Logo Exploration Prompts

## Objetivo

Testar possibilidades de logotipo para *Fragmentos do Amanha* antes de escolher uma direcao final para vetorizacao.

As imagens geradas nesta fase sao referencias visuais. O logo final deve ser redesenhado/vetorizado manualmente para garantir legibilidade, controle tipografico e uso comercial.

## Regras gerais

- Texto preferencial para exploracao: `FRAGMENTOS DO AMANHA`.
- Evitar texto pequeno.
- Evitar slogans.
- Criar composicoes que funcionem em fundo claro e escuro.
- Priorizar silhueta reconhecivel.
- Evitar depender de detalhes excessivos.
- Conectar visualmente viagem no tempo, ruptura, fragmentos e eras historicas.
- Nao usar simbolos reais de marcas existentes.
- Pensar na capa paisagem do concept book: `29.46 x 20.57 cm`.

## Direcao A — Nucleo temporal fraturado

```text
Use case: logo-brand
Asset type: game logo exploration sheet
Primary request: Create a vector-friendly logo exploration for the game title "FRAGMENTOS DO AMANHA".
Subject: a circular temporal core emblem cracked into several floating fragments, suggesting time rupture and sci-fi machinery.
Style/medium: polished game logo concept, clean vector-like shapes, readable silhouette, no photorealism.
Composition/framing: centered emblem above or beside the wordmark, include dark-background and light-background applications, suitable for a 29.46 x 20.57 cm landscape concept book cover.
Lighting/mood: controlled, mysterious, high contrast.
Color palette: graphite, cold cyan, warm copper/orange accent.
Text (verbatim): "FRAGMENTOS DO AMANHA"
Constraints: large readable title only, no small body text, no slogan, no mockup on products.
Avoid: illegible typography, excessive ornament, generic neon cyberpunk, horror logo, existing brand marks.
```
